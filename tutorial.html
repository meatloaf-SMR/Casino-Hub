How to Use the Casino Hub

Opening Games: Scroll through the game cards and click “Play” to load a game in the frame below.

Panic Key: Press Escape at any time to instantly switch to a calculator screen. This also changes the URL to look harmless.

Update Bar: The white bar at the top of the hub shows the latest announcement. You can edit it through updates.txt in your GitHub repo.

Info Page: Shows your local stats, name, and other personalized info saved on your device.

Startup Gate: When you first load the site each day, you’ll be asked if you’re a student or teacher. Students enter normally; teachers get redirected.

Step 1 — Welcome

Welcome to the Casino Hub! This quick walkthrough will show you how everything works so you can jump right in.

Step 2 — Opening Games

Each game is displayed as a card. Click “Play” to load it instantly in the game frame. No new tabs, no clutter.

Step 3 — Panic Key

If you ever need to hide the hub quickly, press Escape. A calculator appears and the URL changes to /calculator.

Step 4 — Updates

The update bar at the top loads text from updates.txt. Edit that file in GitHub to change the announcement.

Step 5 — You're Ready!

You now know how to use the hub. Have fun exploring the games!
